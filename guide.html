🎓 Bienvenue

Voici les étapes pour bien commencer :

  1. Lire le règlement.
  2. Installer les addons (lien vers Steam).

  4. Découvrir les rôles disponibles.
  5. Lire le lore du serveur.

  • Utilise la touche tab pour voir l'overlay Steam.
